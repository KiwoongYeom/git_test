Overview¶

GitHub는 버전 관리 및 협업을 위한 코드 호스팅 플랫폼입니다.

하나의 플랫폼에서 전체 개발 프로세스를 관리 할 수 있습니다.

GitHub는 개발자에게 인기있는 IDE, 지속적인 통합 도구 및 타사 앱과 서비스의 수백 가지를 지원하여 개발 과정에서 개발자들이 사용하던 여러 도구들를 계속 사용할 수 있습니다.

Ability¶

  • Collaboration
  • GitHub Enterprise는 코드 협업 및 평가는 개발 프로세스에 내장되어 있습니다. 당신의 팀은 작업을 공유, 변경 사항을 논의를 한 곳에서 피드백 할 수 있습니다.
  • Security
  • SAML 또는 LDAP 동기화와 한 곳에서 권한을 관리 할 수​​ 있습니다.
  • Hosting Options
  • GitHub Enterprise는 자신의 서버 KVM-또는 사설 클라우드에 VMware, AWS 및 OpenStack을 볼 수 있습니다. 게다가, 당신은 보안 및 규제 요구 사항을 충족하기 위해 인스턴스를 구성 할 수 있습니다.
  • Support
  • GitHub의 뛰어난 글로벌 지원 팀은 주 5 일 24 시간 사용할 수 있습니다. 또한 GitHub Enterprise로 긴급한 문제에 대해 도움을 받을 수 있습니다.

single platform for collaboration¶

팀이 하나의 사무실이나 전 세계에 앉아 협력할 수 있는 GitHub의 워크 플로우가 내장되어 있습니다.

Communicate without friction¶

Pull 요청으로 개발자가 코드에 대한 변경, 공유 아이디어를 제안을 할 수 있게 지원합니다.

깔끔한 타임 라인 스타일의 인터페이스에서 서로 배워 더 나은 협력관계를 야기시킵니다.

Work with transparency¶

하나의 플랫폼에서 모두와 함께 프로젝트에 더 큰 시야를 얻을 수 있습니다.

사용자 이름 또는 URL을 공유 하여 쉬운 대화와 공유를 할 수 있습니다..

Track issues¶

GitHub의 통합 이슈 트래커와 버그 상단에 팀을 유지합니다.

진행 상황 추적 소유권을 할당하고, 마크된 할당 문제를 앞으로 쉽게 이동할 수 있습니다.

모든 개발자가 회사에 존재하는 모든 코드를 찾을 수 있습니다.

기존 코드를 활용하여 시간을 절약할 수 있습니다.

Enterprise-grade security¶

  • 권한 조권 지정 가능
  • 중앙에서 권한 관리
  • 기술적 지원 가능
  • 모니터링 및 대시보드 제공

Extend GitHub¶

GitHub Enterprise는 애플리케이션과 서비스 개발 환경에 대한 사용자 정의 지원합니다.

전체 개발 과정을 보다 효율적으로 작업 할 수 있도록 GitHub를 확장 할 수 있습니다.
