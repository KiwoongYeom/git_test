Overview¶

그룹 채팅 및 비공개 채팅, 파일 공유 및 타시스템 통합을 지원하는 기업용 채팅 도구입니다.

Team chat that’s actually built for business¶

그룹 채팅, 화상 채팅, 화면 공유를 지원하여 IT 팀의 소통을 향상시킵니다.

Run it on everything¶

맥, 윈도우, 아이폰 OS, 안드로이드, 리눅스 그리고 웹에서 모두 지원하여 메시지를 놓치지 않습니다.

Group video chat that works¶

때때로 보여주는 이 설명하는 것보다 쉽습니다. 동료들과 화면 공유를 시작하고 쉽게 프로젝트나 문제 화상으로 이야기 할 수 있습니다.
