Over View¶

고객의 기대 이상에 긍정적인 비즈니스 성과를 위해 모바일 및 웹 테스트를 제공합니다. Keynote 클라우드 기반의 테스트, 모니터링 및 분석 솔루션 등이 디지털 채널을 통해 품질과 성능을 최적화 할 수 있습니다.

Mobile Experience¶

높은 품질로 고객을 놀라게 고성능 모바일 경험을 제공 할 수 있습니다.

Moblie test¶

실제 모바일 장치에서 모바일 앱을 테스트하여 품질을 확인합니다.Keynote 모바일 테스트는 수동 장치 제어와 함께 강력한 테스트 자동화 기능을 제공합니다.

Mobile App Monitoring¶

유일한 100% 클라우드 기반 모바일 애플리케이션 모니터링 솔루션으로 지속적으로 24 시간 동안 실제 캐리어 네트워크를 통해 실제 스마트 폰과 태블릿에서 모바일 앱의 성능과 가용성을 확인합니다.

User Insights¶

모바일 앱, 웹 사이트를 모니터링, 분석하여 서비스의 성능과 가용성을 향상시킬 수 있습니다.

Dynatrace Synthetic Monitoring¶

Keynote Web Monitoring 등 모든 기능을 dynaTrace Web Monitoring에 통합되었습니다.테스트, 측정, 개선, 비교를 제공합니다.

Dynatrace Load¶

Keynote 로드 테스트는 이제 dynaTrace는 로드의 일부입니다.클라우드에서 로드, 웹 및 모바일 애플리케이션의 성능을 테스트합니다.

Performance Indexes¶

Keynote Performance Indexes는 키 웹 및 세계 주요 산업에서 모바일 사이트에 대한 주간, 독립적 인 성능 데이터를 제공합니다.
