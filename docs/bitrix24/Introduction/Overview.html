Overview¶

Bitrix24는 각종 도구를 지원하여 보다 편한 협업 지원합니다.

중앙 집중식 직원 검색 디렉토리¶

중앙 집중식 검색 직원 디렉토리를 생성하지 않고 많은 직원들을 관리 할 수​​ 있습니다.

  • 사용자 정의 직원 프로필
  • 연락처 정보 (전화, 이메일, 소셜 프로필)
  • 채팅, 그룹 채팅, 화상 채팅 지원
  • 기술이나 관심사로 검색
  • 변경 사항 및 타임 라인확인
  • AD / LDAP / NTLM 통합

기업 조직도¶

Bitrix24는 부서, 하위 부서, 상위 부서 그리고 부하 직원과 조직의 계층 구조를 나타내는 상호 작용 다이어그램을 쉽게 만들 수 있습니다.

  • 권한 설정
  • 작업 및 프로젝트관리
  • 다중 분할 (각 부서에 대해 별도의 인트라넷)

뉴스 및 공지 사항¶

  • 회사 차원의 공지 사항
  • RSS를 통해 외부 뉴스 피드 가져 오기
  • 회사 소식

일정 공유¶

  • 부재 차트
  • 요청 작성
  • 출장 요청

지식관리¶

  • 학습 / 교육 시스템
  • 기술 자료
  • 아이디어 관리
  • 위키

문서관리¶

  • 개인 및 공유 문서 : 개인 / 팀 / 회사
  • 문서 동기화와 네트워크 드라이브 매핑 및 데스크톱 응용 프로그램
  • 내부 편집 (Google 문서 도구 / MS 오피스 웹 앱 / MS 오피스)
  • 버전 기록

다른 툴과의 통합¶

Bitrix24는 오픈 API와 함께 제공됩니다. 그래서 사용자 정의하고 당신이 사용하는 다른 인사 관리 및 급여 솔루션, 엔터프라이즈 소셜 네트워크 / 인적 자원 정보 시스템을 통합하기 쉽습니다.
