Overview¶

복잡한 우주 항공 및 방위 산업에서 증가하는 도전적이고 비용이 많이드는 문제를 해결합니다. 종속성, 가시성 및 제어 시스템 공학, 공급망 관리 및 제품 개발 프로세스 등 높은 가치와 매우 복잡한 비즈니스 도메인의 복잡한 네트워크의 이해를 돕는 혁신적인 소프트웨어 응용 프로그램입니다.

essentials¶

BOXARR 플랫폼은 ISM(Inter-dependent Systems Management)을 위한 주요 솔루션입니다.효과적인 모델 시각화, 분석 및 실시간 정보를 최적화하기 위한 조직을 가능하게합니다.각 요소는 여러 사용자 정의 데이터 필드를 지정할 수 있습니다.조직 전체의 기존 데이터 / 신호 채널을 활용하여 거의 모든 소스에서 효율적으로 정보를 수집하는 기업을 가능하게합니다.

modelling paradigm¶

BOXARR 는 간단한 연결로 강력한 모델을 제공합니다.

collaboration¶

BOXARR 모델을 구축하고 가치있는 결과를 만드는 공동 플랫폼입니다.여러 사용자가 동일한 모델 내에서 공동으로 작업 할 수 있습니다.서로 다른 모델 객체 사이의 관계와 영향을 관련 될 수 있도록 최선을 다하고 변경 사항을 실시간으로 모든 사용자에게 전달합니다.

visualisation, navigation and filtering¶

BOXARR은 쉽고 빠르게 대규모 스케일 모델을 시각화 탐색 및 필터링함으로 인간 과기계 인터페이스를 최적화하도록 설계합니다.

functional analysis¶

BOXARR 플랫폼은 강력한 기능 계산과 ‘Boxes& Arrows’모델의 시각적 패러다임을 결합하여 분석할 수 있습니다.기능성 분석은 모델 내의 contructs 데이터에 맞게 하기 위해 사용자들이 설계할 수 있습니다.기능성 분석은 자동 사용자 정의 매핑에서 피벗 테이블 또는 그래프로 주어진 모델 ‘그룹’내에 누적 된 데이터를 표시하는 데 이용 될 수 있습니다.
