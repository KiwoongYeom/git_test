Over View¶

Enterprise Source Code Management. Open Source. Secure. Unified.¶

Centralized control for distributed repositories. Mercurial, Git, and Subversion under a single roof.

Mercurial, Git 및 Subversion과 같은 레파지토리들을 중앙 집중 관리 할 수 있습니다.

Features¶

  • Mercurial, Git & Subversion
  • Intuitive System Management
  • Full Text Search
  • Easy Access Management
  • Online Editor
  • Visual Changelog & Graphlog
  • Large Binary File Support

Unified Tools for Mercurial, Git and Subversion¶

  • 더 나은 코드품질을 위한 팀 공동작업
  • 빠른 인터렉션을 위한 워크플로우 자동화
  • 안전한 소프트웨어 개발을 위한 권한 관리

를 지원하여 더 나은 협원을 지원합니다.

Business Benefits That Pay for Themselves¶

Faster time to value¶

RhodeCode를 사용 하면 코드를 더 빨리 테스트하여 버그 줄이고 회사의 코드 베이스에 걸쳐 모범 사례를 적용 수 있습니다.

Military-grade security¶

방화벽 뒤에서 팀과 작업물을 보호합니다. 같은 플랫폼 안에서 보안등급이 높은 작업물과 함께 고객들의 작업물을 보호하고 공유 할 수 있습니다.

Increased IT ROI¶

로드멥에 활용 할 수 있도록, 더많은 투자를 제공합니다. 새로운 애자일 프로젝트와 기존의 앱 및 팀 분리된 잠금을 해제합니다.

SVN to Git Migration¶

SVN 을위한 지원을 통합했습니다. 앞으로 수년 동안 당신의 SVN 응용 프로그램 및 도구에서 지속적인 값을 추출하는 동안, 깃에 연동하지 않는 경로가 있습니다.

Improved Team Collaboration¶

빠르고 뛰어난 결과를 제공합니다. 프로젝트를 추진할 때에 더 나은 개발자 리소스를 할당 할 수 있는 협업환경에서 소프트웨어를 개발합니다.

Global scale¶

큰 규모의 글로벌 개발 팀은 안전하면서도 성능이 우수한 SCM 솔루션을 필요로 합니다. RhodeCode는 이러한 까다로운 글로벌 기업들의 요구사항을 잘 이행합니다.
