Overview¶

SourceBird는 액세스 위반, 배열과 문자열의 오버런, 메모리 누수, 보안 결함과 연산 오류 등 C/C++ 프로그램에서 버그를 검출하기위하여 100% 자동화된 정적 분석 도구입니다. SourceBird는 대부분의 IDE의 빌드 시스템과 완벽하게 통합되며 개발 초기 단계에서 버그 를 발견하여 비용과 시간을 절약시킵니다.

파일, 프로젝트, 솔루션 단위 수행¶

SourceBird 는 IDE 내에서 컴파일러처럼 실행할 수 있습니다. 이것은 전체 프로젝트 뿐만 아니라 하나의 파일 또는 파일 집합을 분석할 수 있음을 의미합니다. 모든 조작법이 기존 IDE에서 자연스럽게 확장되어 학습곡선을 감소시키고 생산성이 향상됩니다.

시각적인 에러 추적¶

SourceBird는 추상화된 오류 추적기능을 제공합니다. 이 기능은 버그 취약점의 원인에 대한 설명의 역할을 합니다. 모든 추적을 시뮬레이션 할 수 있습니다. 즉, 해당 코드 라인을 인터랙티브 방식으로 강조할 수 있습니다. 버그의 위치를 빠르게 파악하는데 도움이 됩니다.

고급화된 점검 옵션¶

패키지와 코딩 표준을 체크합니다. 예를 들어 간단한 클릭을 통해 MISRA 도는 CERT 보안 검사에 대한 모든 검사를 선택할 수 있습니다. 프로젝트와 관련된 특정 검사를 선택하고 필터링하여 요구사항과 일치하는 부분을 찾아낼 수 있습니다.

다양한 IDE 지원 목록¶

  • Visual Studio 2013, 2008, 2005 버전
  • Eclipse 3.5(Galileo) 이상의 버전
  • QNX 3.1 이상의 버전
  • TI Code Composer 5.0 이상의 버전
  • Windriver Workbench 3.3 이상의 버전
  • Atmel Studio 6.0 과 6.1 버전
  • 이 밖에 Eclipse 기반의 다양한 IDE 지원
