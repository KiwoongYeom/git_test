Overview¶

Conformiq 자동 소프트웨어 테스팅을위한 차세대 솔루션을 제공합니다. 새로운 제품의 요구에 신속하게 대응하며 짧은 시간 동안 힘든 테스트, 실행 스크립트 보수에 필요한 시간을 제거함으로써 신속한 소프트웨어 개발의 요구에 맞는 서비스를 제공합니다. Conformiq 제품의 유지 보수 및 요구 사항은 기존의 테스트 자동화의 문제를 해결하고 더 빠른, 더 나은 테스트하기 위해 조직을 활성화하고 비용을 절감합니다.

Enterprise IT¶

텔레콤, 은행, 금융, 보험, 소매점 및 공공 서비스 등 다양한 분야에서 사용중이며, 높은 가용성과 빠른 채택을 위한 Agile, in-sprint 테스트의 자동화를 가능하게 설계하였습니다.

프로그래밍이 필요없는 시각화된 모델링¶

  • 프로그래밍 경험 무필요
  • 쉬운 모델 웹, 데스크톱 응용 프로그램, 웹에 제공
  • 비슷한 기본 구성요소와 함께 모델링 서비스 제공(WYSIWYC)

Conformiq Designer¶

테스트 자동화 및 통신, 네트워크 장비 업체, 자동화를 포함하여 다양한 산업분야에서의 개발 속도를 높이기 위한 임베디드 소프트웨어 개발에 사용됩니다.end-to-end 테스트 자동화 및 자동화된 테스트 설계를 통하여 높은 소프트웨어 요구 사항과 복잡한 설계 및 시스템 개발을 위해 만들어졌습니다.

뛰어난 품질¶

  • 임베디드 시스템과 엄격한 품질 요구 사항
  • 안전에 중요한 응용프로그램 (ex – 자동차, 항공 우주)
  • 안정적으로 작동하는 수명이 긴 시스템
  • 표준화하거나 규정 승인이 필요한 시스템
  • 높은 보안 요구 사항

유연하고 융통성 있는 다목적 도구¶

  • 다양한 도구에서 표준 UML 다이어그램
  • 미리 만들어진 플러그인으로 쉽게 테스트 실행 프레임워크랑 통합이 가능
  • 표준 Java + 통합 모델링 언어(UML) 프로그래밍 언어와 함께 자신의 모델링 프레임 워크를 개발하는 유연성을 제공

복잡성 제거¶

  • Java + UML 프로그래밍 언어로 복잡한 시스템 동작을 설명
  • 복잡한 시스템 동작을 시각화하고 모듈을 구성
  • 모델의 상태를 읽고 검토의 효율성
  • 유지 보수 및 통신을 향상
  • 관리 및 유지 보수를 개선하는 테스트 최적화
  • 작은 테스트 집합은 빠르게 실행
  • 지능형 테스트 생성 알고리즘은 복잡한 시스템에 관계없이 항상 바로 테스트 생성

임베디드 소프트웨어 및 상태 기반 시스템¶

  • 임베디드 시스템을 테스트하거나 프로토콜과 같은 상태 기반 시스템 및 실시간 제약 조건이 있는 병렬/멀티 스레드 시스템을 테스트
  • 구성 요소 테스트, 복잡한 계산 / 논리 테스트, 충돌 테스트 및 타이밍이 중요한 종속성과 함께 시스템을 이상적으로 테스트

제품 성능의 향상¶

Conformiq Grid는 Conformiq Creator 및 Conformiq Designer에 기능을 추가하는 동반 제품입니다.

더욱 효율적인 성능¶

  • 테스트 생성 후 클라우드로 이동할 때, 더 많은 테스트 생성 프로세스를 제공하여 복잡한 모델에 대한 테스트 생성 프로세스의 속도 향상.
  • 클라우드 / 서버 설치, 분산 처리 요성, 사용자 요구에 따른 공유를 제공
  • 최대 90% 더 효율적인 다중 CPU 코어 및 테스트 생성을 제공
  • 단일 서버 인스턴스는 효율적인 관리 작업 지원
