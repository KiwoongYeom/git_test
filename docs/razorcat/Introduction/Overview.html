Overview¶

TESSY, CTE 및 ITE으로 복잡한 하드웨어 및 소프트웨어 테스트 프로젝트를 관리 할 수​​ 있습니다. 테스트 계획, 테스트 실행, 테스트 평가 및 최종 시험보고 다음 요구 사항 관리에서 시작하여 전체 테스트 프로세스가 지원됩니다.

TESSY¶

Tessy는 C / C++ 임베디드 소프트웨어에 대한 회귀 테스트를 포함한 전체 단위 테스트 사이클을 자동화합니다.쉬운 설치, 직관적으로 볼 수 있는 결과 분석 및 보고에 대한 테스트 설계 및 실행을 통해 프로젝트 설정에서 단위 테스트 워크 플로우를 안내합니다.

Design test cases in the Classification Tree Editor¶

CTM(Classification Tree Method)를 사용하여 테스트 사례를 디자인 하고 CTE의 그래픽 사용자 인터페이스 내에서 CTM을 적용합니다.CTE 내에서 직접 Tessy 테스트 프로세스(ex: 입/출력)에 필요한 값을 쉽게 설정하여 Tessy에 통합 할 수 있습니다.

Edit test data in the Tabular Test Data Editor¶

Test Data Editor(TDE)의 도움으로 테스트 데이터의 편집이 간단합니다.또한 테스트 데이터는 플롯에서 시각화돼어 편한 확인이 가능합니다.

Analyze the test results¶

Test Data Editor에서 그래프 내에서 테스트 결과 시각화하고 테스트 결과를 분석 할 수 있는 범위에 대한 도움말 기능을 제공합니다.

CTE(Classification Tree Editor)¶

Classification Tree Editor (CTE)는 테스트 케이스의 집합을 분류 트리 방법을 적용 할 수 있는 그래픽 편집기입니다.

Use the graphical editor¶

Classification Tree Editor로 ​​트리를 디자인하고. 직관적 인 방법으로 테스트 케이스를 만드는 데 사용되는 값을 설정해 이해하기 쉬운 결과로 트리 요소를 묘사합니다.또한 더 상세한 하위 요소와 하위 트리를 통해 쉬운 탐색, 트리에 대 테스트 케이스 명세 기능과 드래그 & 드롭 및 짧은 키 처리를 간단하게 제공 합니다.

Creating test case specifications¶

테스트 케이스를 쉽게 생성 할 수 있고, 사용자는 체계적인 트리 분류 법에 의해 유도되어 쉬운 테스트를 할 수 있습니다.값을 한번만 입력하고 조합 테이블에 표시함으로써 간단하게 테스트 케이스에 대해 설정 될 수 있습니다.표시된 트리 소자 및 그 값의 설명 텍스트는 테스트 케이스 명세를 생성하기 위해 사용됩니다.

Export test case specifications¶

또한 테스트 게이스를 *.xml, *.html, *.txt, *.m (Matlab 파일)파일로 내보낼 수 있습니다.

ITE(Integrated Test Environment)¶

Integrated Test Environment (ITE)는 복잡한 하드웨어 또는 소프트웨어 테스트 프로젝트를 처리하는 솔루션입니다.다른 문서의 요구 사항과 테스트 케이스 관리, 계획 테스트 및 테스트 캠페인을 연결 하는 기능을 제공 합니다.또한 테스트 개발, 실행 되고 관련 요구 사항을 고려 하여 평가, 보고서에 보고 할 수 있습니다.

Define, manage and link requirements¶

문서의 요구 사항에 대한 기본적인 관리 기능을 제공합니다. 또한 그것은 CVS와 XML 파일을 통해 DOORS에서 요구 사항을 가져올 수 있습니다.요구 사항 및 검증 결과는 XML 및 ReqIF 형식으로 내보낼 수 있습니다.특정 테스트 수단과 V & V 매트릭스와 링크 요구 사항을 정의 할 수 있습니다.

Trace dependent elements¶

종속 요소 (요구 사항, 테스트 사례, 테스트 실행 및 보고서)의 전체 테스트 체인을 언제든지 추적하고 테스트 체인에 있는 모든 요소가 서로 연결 됩니다.종속성을 시각화 하고, 테스트는 테스트 커버 특정 요구 사항, 평가 방법 및 테스트를 수행한 사람을 확인하여 더 나은 테스트를 지원합니다.

Define search queries¶

미리 정의 된 쿼리를 개인 또는 공용 검색 템플릿 중 하나로서 ITE 서버에 저장할 수 있습니다.사용자가 링크 / 종속 요소 및 ITE에서 사용할 수있는 다른 속성을 포함한 거의 모든 요소를​​ 검색 할 수 있기 때문에 특정 정보를 쉽게 찾을 수 있습니다.

Generate test reports¶

PDF에서 보고서를 자동으로 생성하고 테스트 케이스에 대한 상세한 구성 보고서를 생성할 수 있습니다.
